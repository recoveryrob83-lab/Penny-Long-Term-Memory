## Structural Notes

LifeOS HQ is the meeting room, not a ninth project owner.

Chief of Staff HQ is the everyday operational front door and receives department reports.

Life OS Maintenance HQ maintains global GitHub hygiene, shared infrastructure, audits, boot integrity, and reconciliation.

The LifeOS Flow Board is an attention surface, not a replacement for Todoist, Calendar, or GitHub.

The LifeOS Dashboard is locally running and tested with four verified sources: GitHub, Trello, Todoist, and Google Calendar private iCal.

Guarded GitHub auto-sync is limited to clean, strictly-behind fast-forward updates. Gmail and Drive dashboard adapters remain deferred.

Desktop automation is validated across the seven department HQs. The Command Center recognizes LifeOS HQ as an eighth exact destination. Current automation labels for Chief of Staff and Maintenance still use the legacy names until Engineering performs a separately authorized update.

The Automation Command Center supports manual and scheduled runs, but scheduling is not yet production-ready for fully unattended Windows operation.

Office Leaks is publicly operating in a live organic market-test phase.

The Engineering PennyOS humble-beginnings note is historical context only. It does not create a new project, roadmap commitment, or open loop.

Detailed facts, source links, financial records, credentials, private calendar URLs, call notes, event details, and working records belong in their authoritative operational systems.
