# LifeOS HQ Naming Standard

## Purpose

This file is the canonical naming source for top-level LifeOS headquarters across ChatGPT chat titles, automation targets, prompts, dashboards, and current durable-memory references.

## Canonical names

- `LifeOS HQ` — central hub
- `Engineering HQ`
- `Logistics HQ`
- `Wellness HQ`
- `Finance HQ`
- `Main Assistant HQ`
- `Business HQ`
- `Office Leaks HQ`

## Rules

1. Use the canonical name exactly as written for the corresponding ChatGPT conversation title.
2. Use the same canonical name in active automation mappings, current prompts, dashboard labels, and current identity references.
3. Do not add prefixes such as `Chief`, personal-role labels such as `Penny`, or project descriptors such as `Benefits` to the top-level HQ name.
4. Historical notes, archived advisories, and prior commit records may retain the wording that was accurate when they were created.
5. Project folder names remain unchanged unless a separate migration is explicitly authorized. Canonical HQ names are display and identity names, not automatic filesystem rename instructions.

## Noncanonical legacy examples

The following are legacy labels and should not be used for new top-level HQ references:

- `Life OS Hub`
- `Chief Engineering HQ`
- `Chief Logistics HQ`
- `Chief Wellness HQ`
- `Finance Penny HQ`
- `Finance Benefits HQ`
- `Chief Business HQ`
- `Main Assistant Penny`
- `Office Leaks VP Penny / Office Leaks Consulting HQ`
