# Life Logistics HQ Sources

Updated: 2026-07-03

## Source Systems

- GitHub: durable memory map and project state.
- Google Drive: working records and artifacts.
- Todoist: action queue.
- Calendar: timed commitments.
- Gmail: communication evidence.
- Contacts: people lookup.
- Advisory boards: cross-department notices.
- RPR: user-mediated file path when needed.

## Rule

Use pointers and summaries. Do not duplicate detailed records into GitHub by default.
