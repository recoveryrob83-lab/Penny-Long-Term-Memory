# Scheduled Task Memo Template

Use this format when a scheduled task leaves a memo for a department.

```markdown
## MEMO-YYYYMMDD-HHMM — Short Subject

- Date / Time:
- Task ID:
- Task Name:
- Ran As:
- Target Department:
- Status: Completed / Partial / Failed / Needs Review
- Source Files Checked:
- Memo Type: Brief / Itinerary / Advisory Check / Housekeeping / Watch / Other

### Result

Short useful result.

### What Was Already Done

List completed actions.

### Needs Rob / Department Attention

List next actions or say none.

### Errors / Limits

List any access issue, connector issue, or uncertainty.
```
